## 1) 與需求對照（關鍵差異）

### 已符合
- 三種角色：管理員/教練/學員。
- 教練註冊需審核。
- 學員由教練建立，採 OTP 登入。
- 學員不可管理班別、小組、其他學員。
- R&R、通訊錄屬於小組層級。
- CI/CD 與部署健康檢查已建立。

### 未完全符合
- 小組預設首頁應為「追蹤表」：目前仍為小組總覽。
- 小組六大模組尚未完整（目前僅 R&R、通訊錄先有基礎版）。
- 追蹤表權限拆分（教練管理項目、學員僅勾選）尚未實作。
- 非所屬班別/小組唯讀瀏覽尚未完整實作。

## Phase UX-0：介面修整（先完成）
- [x] UX-1. 登入表單視覺重整：
  - 按鈕與輸入欄位垂直節奏（間距 12-16px）、按鈕區塊獨立留白。

- [x] UX-2. 身份選擇頁重整：
  - 改雙欄身份選擇卡（學員/教練），加入中線分隔、圓形 icon 區與可辨識主標。

  - 手機版改縱向堆疊，維持可點擊區與字級。
- [x] UX-3. 學員管理清單 DataTable 化：

  - 單筆操作改 icon：綠色打勾（儲存）、黃色叉叉（取消）、紅色垃圾桶（刪除）。
